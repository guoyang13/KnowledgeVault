# 系统书单与权威资料

## 入门：知识图谱中文书

**《知识图谱：概念与技术》 肖仰华等**

用途：建立中文语境下的整体框架，理解知识图谱是什么、怎么构建、怎么存储、怎么查询、怎么应用。

重点阅读：

- 知识表示
- 知识建模
- 实体、关系、属性
- 图谱构建
- 查询与推理
- 行业知识图谱实践

## 标准技术：A Semantic Web Primer

**A Semantic Web Primer**  
作者：Grigoris Antoniou、Paul Groth、Frank van Harmelen、Rinke Hoekstra

用途：系统理解语义网标准技术栈。

重点掌握：

```text
RDF → RDFS → OWL → SPARQL → Linked Data
```

## 实战建模：Semantic Web for the Working Ontologist

**Semantic Web for the Working Ontologist: Effective Modeling for Linked Data, RDFS, and OWL**  
作者：Dean Allemang、Jim Hendler、Fabien Gandon

用途：从“知道 RDF/OWL 是什么”进阶到“知道怎么建一个靠谱本体”。

重点掌握：

- RDFS / OWL 建模能力
- 类层级设计
- 属性设计
- domain / range 的真实语义
- 推理对建模的影响
- 常见建模误区

## 查询专项：Learning SPARQL

**Learning SPARQL, 2nd Edition**  
作者：Bob DuCharme

用途：专门练 SPARQL 查询。适合边读边写查询。

## 本体工程经典：Ontological Engineering

**Ontological Engineering: with examples from Knowledge Management, e-Commerce and the Semantic Web**  
作者：Asunción Gómez-Pérez、Mariano Fernández-López、Oscar Corcho

用途：理解本体开发过程、生命周期、方法论、工具和语言。

重点掌握：

- 如何从业务领域抽象概念
- 如何设计类、属性、关系
- 如何复用已有本体
- 如何维护本体生命周期
- 如何做本体工程方法论

## 最小书单

如果只读三本：

1. **《知识图谱：概念与技术》**：建立知识图谱 / 本体的大图景。
2. **A Semantic Web Primer**：掌握 RDF / OWL / SPARQL 标准栈。
3. **Semantic Web for the Working Ontologist**：学习本体建模实践。

如果偏工程落地，可以把第三本提前读，边读边做 [[本体模型/12-供应链本体Mini项目]]。
