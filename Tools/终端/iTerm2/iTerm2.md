iTerm2 是 Mac 上公认的神级终端工具，比系统自带的终端强大得多。要玩转 iTerm2，可以从以下几个维度掌握它的“黑科技”技巧：

### 1. 效率翻倍的分屏功能 (Split Panes)
这是 iTerm2 最实用的功能。你可以在一个窗口里同时看代码、运行程序、操作 Git。
*   **垂直分屏**：`Command (⌘)` + `D`
*   **水平分屏**：`Command (⌘)` + `Shift (⇧)` + `D`
*   **切换分屏**：`Command (⌘)` + `[` 或 `]`（或者用 `⌘` + `方向键`）
*   **调整分屏大小**：`Command (⌘)` + `Option (⌥)` + `方向键`

### 2. 全局热键呼出 (Hotkey Window)
想象一下，无论你在用浏览器还是写代码，按下快捷键终端就从屏幕上方滑出，再按一下就消失，极其方便。
*   **设置方法**：
    1.  `Preferences` -> `Profiles`。
    2.  点击左下角 `+` 新建一个名为 "Hotkey Window" 的 Profile。
    3.  点击该 Profile 的 `Keys` 标签页，勾选底部的 `Create a Dedicated Hotkey Window`。
    4.  设置一个你喜欢的快捷键（比如 `Option` + `Space`）。

### 3. 搜索与自动补全
*   **查找输出内容**：`Command (⌘)` + `F`（支持正则表达式，找报错日志神器）。
*   **命令自动补全**：输入命令开头，按下 `Command (⌘)` + `;`。它会弹出一个列表，展示你最近输入过的相关命令。
*   **历史记录查询**：`Command (⌘)` + `Shift (⇧)` + `H`（会弹出一个浮窗显示你所有输入过的命令）。

### 4. 智能选中与复制
*   **自动复制**：在 iTerm2 中，你**只要选中**一段文本，它就自动复制到剪贴板了，不需要按 `⌘+C`。
*   **智能选中**：双击一个路径、一个网址或一个变量名，它会智能识别并选中。
*   **无缝粘贴**：点击鼠标中键（或者如果设置了的话）直接粘贴。

### 5. 标记与快速跳转 (Marks)
当你运行了很长的命令（比如跑了几千行日志），想回到命令开始的地方：
*   **设置标记**：在你运行命令时按下 `Command (⌘)` + `Shift (⇧)` + `M`。
*   **快速跳转**：按下 `Command (⌘)` + `Shift (⇧)` + `J`。它能让你瞬间跳回上一个命令的起始行。

### 6. 广播输入 (Broadcast Input)
如果你同时开了好几个分屏，想在所有窗口里同时输入同一条命令（比如在多个服务器同时拉取 Git 代码）：
*   **开启/关闭**：`Command (⌘)` + `Option (⌥)` + `I`。
*   *注意：开启后，你在当前窗口打字，所有窗口都会同步输入。*

### 7. 颜值配置（配合 Oh My Zsh）
iTerm2 本身只是外壳，灵魂在于主题：
1.  **颜色方案**：去 [iTerm2-Color-Schemes](https://github.com/mbadolato/iTerm2-Color-Schemes) 下载，推荐 `Solarized Dark` 或 `Dracula`。
2.  **Shell 集成**：点击菜单栏 `iTerm2` -> `Install Shell Integration`。安装后，每一行命令左侧会出现一个蓝点，点击蓝点可以快速选中输出内容、查看运行状态。

### 8. 剪贴板历史 (Paste History)
按下 `Command (⌘)` + `Shift (⇧)` + `V`。它会显示你最近复制过的 20 条记录，你可以上下移动选择要粘贴哪一条。

### 9. 撤销关闭 (Undo Close)
如果不小心把正在运行程序的窗口关掉了：
*   **撤销**：按下 `Command (⌘)` + `Z`。iTerm2 会在几秒钟内允许你撤销关闭操作。

### 💡 建议你的第一步配置：
既然你在处理 Git 项目，请务必安装 **Oh My Zsh**。
1.  安装后，iTerm2 底部的状态栏或每一行开头都会显示：
    *   **当前分支名** (如 `main`)。
    *   **状态图标** (❌ 代表有文件没提交，✅ 代表仓库是干净的)。
2.  **推荐插件**：
    *   `zsh-autosuggestions`：它会像浏览器搜索框一样，灰度预测你想输入的下一条命令，按方向键 `→` 即可补全。
