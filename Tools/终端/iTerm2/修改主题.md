修改 iTerm2 的主题通常包含两个层面：一个是**外观颜色方案（Color Scheme）**，另一个是**终端提示符样式（Shell Theme）**。

为了达到最完美的效果，建议两部分都配置。以下是具体操作步骤：

### 1. 修改颜色方案（iTerm2 软件本身颜色）

这是改变背景色、文字颜色和各种命令颜色的最直接方式。

1.  **使用内置预设**：
    *   打开设置：`Command (⌘)` + `,`。
    *   依次点击：`Profiles` -> `Colors` -> 右下角的 `Color Presets...`。
    *   在下拉菜单中选择一个喜欢的，比如 `Solarized Dark` 或 `Dark Background`。

2.  **导入第三方精美主题（推荐）**：
    如果你觉得内置的不好看，可以去最大的主题库 [iTerm2-Color-Schemes](https://github.com/mbadolato/iTerm2-Color-Schemes/tree/master/schemes) 下载。
    *   **下载主题**：去该网站找到喜欢的名字（比如 `Dracula` 或 `Snazzy`），右键保存 `.itermcolors` 文件。
    *   **导入**：在刚才的 `Color Presets...` 下拉框中选择 `Import...`。
    *   **应用**：导入后，再次点击下拉框选中它。

---

### 2. 修改 Shell 主题（Oh My Zsh 样式）

这决定了你终端每一行开头的样式（是否带图标、是否显示路径等）。

#### 情况 A：如果你使用的是 Powerlevel10k（之前配置过的图标主题）
如果你想改变它的样式（比如从“彩虹色”换成“简约黑白”）：
*   在终端直接输入：
    ```bash
    p10k configure
    ```
*   这会重新进入那个交互式的问答界面，你可以换一种完全不同的布局。

#### 情况 B：你想换成其他的 Oh My Zsh 内置主题
Oh My Zsh 自带了上百种主题，你可以随时切换：
1.  打开配置文件：
    ```bash
    nano ~/.zshrc
    ```
2.  找到这一行：`ZSH_THEME="powerlevel10k/powerlevel10k"`。
3.  把它改成别的主题名字（比如经典的 `robbyrussell` 或 `agnoster`）：
    ```bash
    ZSH_THEME="robbyrussell"
    ```
4.  保存退出（`Ctrl+O` 回车，`Ctrl+X`），然后输入 `source ~/.zshrc` 生效。

---

### 3. 让窗口变透明和毛玻璃效果 (进阶外观)

为了让 iTerm2 看起来更高级，可以开启透明度和模糊效果：
1.  打开设置：`Cmd (⌘)` + `,`。
2.  进入 `Profiles` -> `Window` 标签页。
3.  **Transparency (透明度)**：向右滑动调整（建议 15% 左右）。
4.  **Blur (模糊)**：勾选并调整（建议 15-20），这样背景就会有毛玻璃感。

---

### 4. 开启状态栏 (Status Bar)

iTerm2 底部可以显示 CPU、内存使用情况和当前路径：
1.  进入 `Profiles` -> `Session`。
2.  勾选底部的 **Status bar enabled**。
3.  点击旁边的 **Configure Status Bar**。
4.  把你想看的东西（如 `CPU Utilization`、`Memory Usage`、`Git State`）拖到上面的方框里。
5.  在 `Appearance` -> `Status Bar` 还可以调整它的位置（顶部或底部）。

---

### 💡 推荐的主题组合：
*   **极致极客风**：Color Scheme (Dracula) + Oh My Zsh (Powerlevel10k) + 透明度 20%。
*   **简约现代风**：Color Scheme (Snazzy) + Oh My Zsh (Pure) + 无边框模式。

**你现在已经配置好了 Powerlevel10k 吗？如果是，建议直接运行 `p10k configure` 尝试不同的内置样式。**
