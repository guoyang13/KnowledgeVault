`nano` 是终端里最简单、最适合新手的文本编辑器。比起复杂的 `vim`，`nano` 的操作非常直观，所有快捷键都直接显示在屏幕底部。

以下是使用 `nano` 的核心指南：

### 1. 如何打开文件
在终端输入 `nano` 后面跟上文件名。如果文件不存在，它会创建一个新文件。
```bash
nano ~/.zshrc
```

### 2. 基础操作（打字与移动）
*   **输入**：直接像记事本一样打字即可。
*   **移动光标**：使用键盘的 **上下左右方向键**（鼠标点击在 `nano` 里通常无效）。
*   **翻页**：使用 `Ctrl + V`（下一页）和 `Ctrl + Y`（上一页）。

---

### 3. 最重要的快捷键（底部提示栏）
注意：底部的符号 **`^` 代表键盘上的 `Control` 键**（Mac 上也是用 Control，不是 Command）。

*   **保存文件 (Write Out)**：**`Control + O`**
    *   按下后，它会提示 `File Name to Write: ...`，直接**按回车键 (Enter)** 确认即可。
*   **退出 (Exit)**：**`Control + X`**
    *   如果你修改了内容还没保存，它会问你 `Save modified buffer?`。
    *   输入 **`Y`** 表示保存并退出。
    *   输入 **`N`** 表示放弃修改并退出。
*   **搜索 (Where Is)**：**`Control + W`**
    *   输入你想找的关键单词（比如 `ZSH_THEME`），按回车，光标会自动跳过去。

---

### 4. 进阶快捷键（剪切与粘贴）
*   **剪切整行 (Cut Text)**：**`Control + K`**
*   **粘贴 (Uncut Text)**：**`Control + U`**
    *   你可以连续按好几次 `Ctrl + K` 删掉几行，然后换个位置按一次 `Ctrl + U` 把它们全粘出来。

---

### 5. 实际演练：修改 `.zshrc`
如果你要修改 Zsh 配置，流程是这样的：

1.  输入 `nano ~/.zshrc` 进入。
2.  用方向键找到你要改的那一行。
3.  直接删掉旧内容，打入新内容。
4.  按下 **`Control + O`**，然后按 **回车**（保存）。
5.  按下 **`Control + X`**（退出）。
6.  **重要提示**：修改完配置后，别忘了运行 `source ~/.zshrc` 让修改立即生效。

### 总结口诀：
*   **Ctrl + O**：保存。
*   **Ctrl + X**：退出。
*   **Ctrl + W**：找东西。
*   **Ctrl + K/U**：删行/贴行。

**在配置 iTerm2 时，如果你怕改乱了，可以直接按 `Ctrl + X` 然后选 `N` 退出，文件就保持原样不会变。**
