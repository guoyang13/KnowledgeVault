# Antigravity

> Google 面向 AI 编程工作流的 Agent 产品入口，适合作为 Coding Agent 生态观察对象。

- **形态**：面向软件工程任务的 AI Agent 工作台。
- **关注点**：代码库理解、任务执行、权限边界、与现有 IDE/仓库流程的衔接。
- **比较维度**：与 Cursor、Codex、Claude Code、Devin 等工具对比 agent harness 与工程闭环能力。
