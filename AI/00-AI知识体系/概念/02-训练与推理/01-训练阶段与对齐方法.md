# 训练阶段与对齐方法

> 现代 LLM 的训练通常分四步：Pretraining → SFT → Preference Alignment → 任务专项 RL。对齐阶段决定了模型"好不好用"。

## 训练阶段
1. **Pretraining**：海量文本自监督，目标是 next-token prediction。
2. **Continued Pretraining / Mid-training**：注入新数据、新模态、新语种、新代码语料。
3. **SFT（Supervised Fine-Tuning）**：用高质量"指令-回答"对教模型遵循指令。
4. **Preference Alignment**：让模型偏好"人类更喜欢的回答"（见下方）。
5. **任务专项 RL**：[[RLVR]]、Agentic RL 等，强化特定能力。

## 对齐方法
- **RLHF（Reinforcement Learning from Human Feedback）**
  - 经典三段式：SFT → 训 Reward Model → PPO 微调。
  - 强但工程复杂、训练不稳。
- **DPO / IPO / KTO / SimPO**
  - 直接用偏好对（chosen / rejected）优化，跳过 Reward Model。
  - 工程简单、效果接近 RLHF，已成新基线。
- **RLAIF（RL from AI Feedback）**
  - 用强模型代替人类打分，规模化更容易。
- **Constitutional AI**（Anthropic）
  - 给模型一套"宪法"原则，让它自我批评并修正。
- **RLVR**：见 [[RLVR]]，用可验证奖励训推理能力。

## 实践要点
- 数据 > 算法：SFT 数据质量与多样性比换算法重要得多。
- 偏好数据建议 prompts 来自真实使用分布。
- 多阶段叠加：SFT → DPO → 任务 RL 是当前事实标准。
