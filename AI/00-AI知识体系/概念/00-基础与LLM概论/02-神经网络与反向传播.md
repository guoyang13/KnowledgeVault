# 神经网络与反向传播

> 理解 LLM 的**最小数学栈**：可微分的函数堆叠 + 用梯度下降从数据里学参数。Transformer 再复杂，训练仍依赖这一套。

## 从神经元到网络

- **神经元**：\(y = \sigma(w^\top x + b)\)，\(\sigma\) 为非线性**激活函数**（ReLU、GELU、SiLU 等）
- **层**：把许多神经元并排/堆叠；**深度网络** = 多层非线性变换
- **前向传播**：输入 → 逐层计算 → 得到预测（对 LLM 即下一个 token 的 logits）

## 损失函数（Loss）

衡量「预测」与「目标」的差距；训练就是**最小化损失**。

| 场景 | 典型损失 |
|------|----------|
| LLM 预训练 / SFT | **交叉熵（Cross-Entropy）**：对每个位置的真实下一 token，惩罚模型给出的错误概率 |
| 回归 | MSE |
| 分类 | 交叉熵（多类） |

语言模型里：位置 \(t\) 的真实 token 为 \(y_t\)，模型输出分布 \(\hat{P}(\cdot \mid x_{<t})\)，损失约为 \(-\log \hat{P}(y_t \mid x_{<t})\)，全序列求平均。

## 反向传播（Backpropagation）

- **链式法则**：从损失 \(L\) 往回算每一层参数对 \(L\) 的梯度 \(\partial L / \partial w\)
- **计算图**：框架（PyTorch、JAX）自动构建图并求导
- **批量训练**：每次用一小批（mini-batch）样本估计梯度，更新参数

## 优化器（Optimizer）

在梯度方向上更新权重，常用：

- **SGD + Momentum**
- **Adam / AdamW** —— LLM 预训练与微调的事实标准之一
- **学习率调度**：warmup、cosine decay 等，影响收敛与稳定性

## 与 LLM 的对应关系

| 基础概念 | 在 LLM 中 |
|----------|----------|
| 参数 \(w\) | 数十亿～千亿权重（Embedding、Attention、FFN） |
| 前向 | 一次 forward 得到整段 logits |
| 损失 | 因果语言建模的 token 级交叉熵（通常只算「预测下一 token」的位置） |
| 反向 | 全参数梯度；规模极大 → 需分布式训练、混合精度、梯度检查点 |
