# 多模态模型（Multimodal / Omni）

> 原生支持文本 + 图像 + 音频 + 视频的统一模型，不再是"文本模型 + 单独的视觉/语音模块"。

## 关键点
- **Native Multimodal**：输入输出共享同一 token 空间，相比"视觉编码器拼接"方案泛化更好。
- **Vision**：ViT / Sigmoid-Loss ViT（SigLIP）→ 多分辨率切片 → 任意尺寸图像。
- **Audio**：编码为离散 token（Whisper-style 或 codec token）。
- **Video**：抽帧 + 时间编码 / 视频 token。
- **Any-to-Any / Omni**：图文音视频任意输入输出（GPT-4o、Gemini）。
- **Multimodal Reasoning**：推理模型 + 多模态（看图做几何题、看截图操作 GUI）。

## 代表模型
- **GPT-4o / GPT-5**、**Gemini 2/3**（Native multimodal 标杆）
- **Claude 3.5/4 Vision**
- **Qwen-VL / Qwen2.5-Omni**、**InternVL**、**LLaVA**
