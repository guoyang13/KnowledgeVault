# 推理模型（Reasoning Model）

> 把"思考过程（thinking / chain-of-thought）"作为一类显式训练目标的模型，用更多推理 token 换更高准确率。

## 关键点
- **核心思想**：不只是 prompt 时让模型 CoT，而是**训练阶段就奖励高质量的思考链**。
- **训练秘诀**：大量 [[RLVR]]（可验证奖励 RL）+ 数学/代码/逻辑题，配合 [[PRM与ORM]]。
- **Thinking Token / Reasoning Token**：模型在输出最终答案前会先生成大段思考，思考内容可见或隐藏。
- **Hybrid Reasoning**：同一模型可在"快答 / 深思"间切换，通过 `reasoning_effort`、`thinking_budget` 控制。

## 代表模型
- **OpenAI o1 / o3 / o4 / GPT-5 thinking**
- **Anthropic Claude 3.7 / 4 Sonnet/Opus（Extended Thinking）**
- **DeepSeek-R1 / R1-Distill**
- **Gemini 2.5 / 3 Thinking**
- **Qwen QwQ / Qwen3-Thinking**

## 应用边界
- **适合**：数学、代码、复杂规划、Agent 长任务。
- **不适合**：简单对话、低延迟场景（思考 token 很贵且慢）。
